## üöÄ Quick Start

1. Copy `5_Symbols/chaptermarkers.lua` or `5_Symbols/fill_last_marker.lua` to:
   - **Windows:** `%APPDATA%\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Edit\`
   - **macOS:** `~/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Edit/`
2. Open DaVinci Resolve, load your project, and open a timeline.
3. Run the script from **Workspace ‚Üí Scripts**.
